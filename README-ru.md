# Описание

Плагин ExteraGram который защищает от открытия пасхалок AyuGram _(tg://ayu/... train, xiaomi, relax, augh, pipe, saul, komaru, lobster, tg://xiaomi, tg://nya)_

_Некоторые пасхалки можно увидеть в [@ayugram_easter](https://t.me/ayugram_easter)_

# Требования
- AyuGram версии 20250612 и выше (на базе ExteraGram 11.12.0 и выше)

# Как это работает

Плагин перехватывает ссылки с формата `tg://ayu/...`. Он проверяет их по заранее заготовленному списку пасхалок. Если ссылка ведет к пасхалке, плагин блокирует её открытие и показывает уведомление.
